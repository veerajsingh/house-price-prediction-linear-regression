\# Bengaluru Property Price Prediction



This project predicts property buying prices in Bengaluru using Linear Regression.



The model uses:



\- Location

\- Total Area (Sqft)

\- Bedrooms (BHK)

\- Bathrooms



to estimate the expected property price.



\---



\## Features



\- Data cleaning and preprocessing

\- Linear Regression model training

\- House price prediction

\- Interactive Gradio UI

\- Dark themed modern website interface

\- Property price output in Lakhs and Crores

\- Graphs and visualizations

\- Model saving using Pickle



\---



\## Dataset



Dataset used:



`bengaluru\_house\_prices.csv`



Important columns used:



\- location

\- total\_sqft

\- bath

\- size

\- price



\---



\## Technologies Used



\- Python

\- Pandas

\- NumPy

\- Matplotlib

\- Scikit-learn

\- Gradio

\- Pickle



\---



\## Project Workflow



1\. Upload dataset

2\. Clean missing values

3\. Extract BHK values

4\. Convert total\_sqft values

5\. Train Linear Regression model

6\. Evaluate model performance

7\. Save trained model

8\. Build Gradio UI

9\. Predict property prices
